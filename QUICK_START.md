# 🍽️ DineVerse - Quick Start (One-Page Cheat Sheet)

## For Café Owners

---

## 📝 Step 1: Register (5 min)
Go to: `yourapp.com/owner/register`

| What | How |
|------|-----|
| Café Name | "The Coffee House" |
| Slug | "the-coffee-house" (auto-generated) |
| Email | your@email.com |
| Password | 6+ characters |

**Result:** Customer link = `yourapp.com/cafe/the-coffee-house`

---

## 🎨 Step 2: Add Logo & Cover (5 min)
Go to: **⚙️ Profile** (left sidebar)

- Upload **Logo** (square, 400×400px)
- Upload **Cover** (wide, 1600×900px)
- Fill description, phone, address
- Click **Save**

---

## 🍽️ Step 3: Build Menu (15 min)
Go to: **🍽️ Menu** (left sidebar)

### 3.1 Add Categories (Optional)
Click **+ Add Category**
- Momos
- Pizza
- Beverages
- Desserts

### 3.2 Add Items
Click **+ Add Item** for each dish:

```
🟢 VEG / 🔴 NON-VEG    ← Choose one
↓
Category → (Momos, Pizza, etc.)
↓
Item Name → (Veg Steamed Momos)
↓
Price → (₹80)
↓
Upload Photo → (600×600px)
↓
✓ Available for Order
↓
SAVE
```

**Repeat for all items** (at least 10 to start)

---

## 📱 Step 4: Share Link (1 min)
Your customer link:
```
yourapp.com/cafe/your-slug
```

**Share via:**
- ✅ WhatsApp to customers
- ✅ Print on menus/receipts
- ✅ Generate QR code: https://qr-server.com/api/qrcode?size=300x300&data=YOUR_LINK
- ✅ Google Business profile
- ✅ Facebook/Instagram

---

## 📦 Step 5: Receive Orders (During Service)
Go to: **📋 Orders** (left sidebar)

**When customer orders:**
1. 🔔 **BEEP BEEP BEEP!** (triple notification)
2. 📬 Toast notification appears
3. 📋 Order shows at top (real-time, NO REFRESH!)

**Manage order:**
```
⏳ Pending
  ↓
✅ Click "Mark as Confirmed"
  ↓
👨‍🍳 Click "Mark as Preparing"
  ↓
🍽️ Click "Mark as Served" (DONE!)
```

---

## 📊 Step 6: Check Dashboard
Go to: **📊 Dashboard** (left sidebar)

**See:**
- 📋 Today's order count
- 💰 Today's revenue
- ⏳ Pending orders
- 👨‍🍳 Preparing orders
- 🏆 Top items this week

## 🎯 Daily Checklist

### ☕ Morning
- [ ] Open Orders page
- [ ] Go to Menu → Update available items
- [ ] Toggle items ON/OFF based on stock

### During Service (4-5 PM - 10 PM)
- [ ] Keep Orders page open
- [ ] Listen for beep notifications
- [ ] Update order status immediately
- [ ] Confirm → Preparing → Served

### Evening
- [ ] Check Dashboard → Today's revenue
- [ ] Check top items sold
- [ ] Plan tomorrow's menu

---

## 🚨 Quick Tips

| Issue | Fix |
|-------|-----|
| No notification sound | Click the page, check volume, refresh |
| Order not updating | Refresh page, check internet |
| Can't upload images | Use PNG/JPG, < 5MB, check connection |
| Customer can't see menu | Verify slug, check items available, check items exist |
| Notification too loud | System volume → turn down |

## 🔗 Customer Flow (Reference)

```
Customer opens yourapp.com/cafe/your-slug
↓
Enter name + table number
↓
View menu (🟢 Veg / 🔴 Non-Veg filters)
↓
Add items to cart
↓
Review bill
↓
Place order
↓
Get receipt with order #
↓

```
